# ✅ Версия 1.0.2 - Критическое исправление

## 🐛 Исправлена критическая ошибка

### Проблема:
Собранные .bin файлы **не загружались в 1С**. Плагин не обновлял адреса в заголовке файла при изменении размера кода.

### Решение:
Теперь плагин **правильно обновляет адреса** в заголовке bin файла:
- Для Format 1: обновляет адрес в блоке 4
- Для Format 2: обновляет адрес в блоке 3

### Результат:
✅ Собранные .bin файлы **успешно загружаются в 1С**  
✅ Код извлекается корректно  
✅ Изменения сохраняются
