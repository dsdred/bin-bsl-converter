# ✅ Версия 1.0.2 - Финальная (только извлечение)

## 📝 Изменения

### Удалена функция сборки BSL → BIN

**Причина**: Сборка bin файлов работает нестабильно из-за сложности формата.

**Решение**: Плагин теперь фокусируется только на **извлечении** модулей из .bin в .bsl.

### Что осталось:

✅ **Извлечение модуля из .bin в .bsl**
- Правый клик на .bin файле → "BIN → BSL: Извлечь модуль"
- Создаётся папка с именем файла и .bsl файл внутри

✅ **Пакетная обработка папки**
- Правый клик на папке → "BIN → BSL: Извлечь из папки"
- Обрабатывает все .bin файлы рекурсивно

✅ **Множественный выбор файлов**
- Через палитру команд можно выбрать несколько .bin файлов

### Что удалено:

❌ Команда "BSL → BIN: Собрать модуль"
❌ Функция insertModuleIntoBin
❌ Функция buildBinFile

---

## 🎯 Новый workflow:

1. **Выгрузите** конфигурацию 1С в файлы
2. **Извлеките** модули из .bin в .bsl через плагин
3. **Анализируйте** и изучайте код в VS Code
4. **Вносите изменения** напрямую в 1С через Конфигуратор
