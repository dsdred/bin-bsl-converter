# BIN-BSL Converter

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Плагин для VS Code для работы с файлами Form.bin и Module.bsl из конфигураций 1С.

## Описание

Плагин предназначен для работы с **конфигурациями 1С на обычных формах**. 

Позволяет:
- **Извлекать текст модулей** из бинарных файлов форм (Form.bin) для удобной доработки в VS Code
- **Конвертировать текст обратно в .bin** после внесения изменений
- Работать с подсветкой синтаксиса, поиском и другими возможностями современного редактора

Это особенно полезно при работе со старыми конфигурациями 1С, где модули форм хранятся в бинарном формате и недоступны для прямого редактирования.

## Возможности

- **Извлечение модуля из .bin в .bsl** - извлекает текст модуля формы из бинарного файла, создаёт папку с именем файла и помещает туда .bsl файл для редактирования
- **Сборка .bin из .bsl** - конвертирует отредактированный текст модуля обратно в бинарный формат .bin
- **Пакетная обработка** - обработка всех .bin файлов в папке рекурсивно
- **Поддержка множественного выбора** - возможность выбрать и обработать несколько файлов одновременно

### Типичный сценарий использования:

1. Выгрузите конфигурацию 1С в файлы (через Конфигуратор)
2. Извлеките модули форм из .bin файлов в .bsl
3. Редактируйте код в VS Code с подсветкой синтаксиса и автодополнением
4. Соберите изменённые .bsl обратно в .bin
5. Загрузите конфигурацию обратно в 1С

## Использование

### Извлечение из .bin файла

1. Правый клик на .bin файле в проводнике
2. Выбрать "BIN → BSL: Извлечь модуль"
3. Будет создана папка с именем файла и в ней .bsl файл

### Сборка в .bin файл

1. Правый клик на .bsl файле в проводнике
2. Выбрать "BSL → BIN: Собрать модуль"
3. Будет создан .bin файл в родительской папке

### Извлечение из папки

1. Правый клик на папке в проводнике
2. Выбрать "BIN → BSL: Извлечь из папки"
3. Все .bin файлы в папке и подпапках будут обработаны

## Структура файлов

```
ИмяФайла.bin          → Исходный файл
ИмяФайла/             → Создаётся папка
  ИмяФайла.bsl        → Извлечённый модуль
```

## Установка

### Способ 1: Из релиза (рекомендуется)
1. Перейдите на [страницу релизов](https://github.com/dsdred/bin-bsl-converter/releases)
2. Скачайте файл `bin-bsl-converter-1.0.0.vsix`
3. В VS Code: `Extensions` → `...` → `Install from VSIX`
4. Выберите скачанный файл

### Способ 2: Из исходников
```bash
git clone https://github.com/dsdred/bin-bsl-converter.git
cd bin-bsl-converter
npm install
npm install -g @vscode/vsce
vsce package
code --install-extension bin-bsl-converter-1.0.0.vsix
```

### Способ 3: Ручная установка
1. Скопировать папку плагина в `.vscode/extensions/`
2. Перезапустить VS Code
3. Плагин готов к работе

## Требования

- VS Code 1.60.0 или выше
- Node.js (для разработки)
- Конфигурация 1С на обычных формах, выгруженная в файлы
